# ResNet18 花卉分类项目

##项目成果
- **测试准确率**: 84.13%
- **最佳验证准确率**: 85.98%
- **训练准确率**: 66.1%
- **训练时间**: 31.2分钟
- **模型大小**: 44.7 MB

##项目简介
使用 PyTorch 和预训练的 ResNet18 模型，实现对5种常见花卉的图像分类。本项目展示了完整的深度学习工作流程，从数据准备到模型训练、评估和结果分析。
